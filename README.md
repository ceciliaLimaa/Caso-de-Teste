# 📋 Desafio QA - Documentação e Casos de Teste

Este projeto foi desenvolvido como um desafio técnico de Quality Assurance com foco em documentação de problemas, elaboração de cenários de teste e melhoria contínua da qualidade de software.

## 🎯 Objetivos

- Documentação de defeitos
- Análise de requisitos
- Criação de casos de teste
- Cenários positivos, negativos e de borda
- Planejamento de qualidade
- Análise de riscos

## 📌 Atividades Realizadas

### Documentação de Problema

Análise de um defeito relacionado à validação do campo e-mail durante o cadastro de usuários, incluindo:

- Impactos para usuário e negócio
- Passos para reprodução
- Resultado esperado
- Sugestões de melhoria

### Casos de Teste

Criação de cenários para uma funcionalidade de agendamento online contemplando:

- Cenários positivos
- Cenários negativos
- Casos de borda

### Plano de Melhoria

Proposta de ações para reduzir bugs em produção e fortalecer a cultura de qualidade dentro do time.

## 🛠️ Competências Demonstradas

- Test Design
- Bug Report
- Testes Funcionais
- Análise de Requisitos
- Análise de Riscos
- Quality Assurance
- Melhoria Contínua
